# 🎨 Colour Pelleting

## Overview
Color Palette Generator is a design-focused project that helps create and customize aesthetically pleasing color schemes for creative, UI/UX, and web design projects. It enables users to explore harmonious color combinations that enhance visual consistency and user experience.

## Features
- 🎨 Generate custom color palettes
- 🌈 Create complementary and analogous color schemes
- 🖌️ Easy-to-use interface
- 💻 Suitable for UI/UX, web, and graphic design
- 📋 Copy HEX and RGB color codes

## Technologies Used
- HTML
- CSS
- JavaScript

## Future Enhancements
- AI-powered palette suggestions
- Export palettes in multiple formats
- Dark/Light mode support

This project uses K-Means Clustering to analyze an image, identify the five most dominant colors, calculate the percentage of each color, visualize the results, and generate a final annotated image showing the extracted color palette. It is useful for applications in graphic design, UI/UX, branding, image analysis, and color palette generation.
